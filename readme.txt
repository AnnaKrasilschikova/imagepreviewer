Тестовое задание реализовано для позиции Frontend-разработчика.
Одно из требований задания - размещение приложения на одном из облачных серверов. 
Прошу уточнить, является ли необходимым условием для проекта размещение его на сервере?
Приложение функционирует в рамках одной сессии до обновления страницы.
В качестве хранилища информации используется массив на JavaScript.
В приложении использован примерный набор изображений, с помощью которого реализована бесконечная прокрутка, 
	и который можно скорректировать добавив или удалив содержимое тега "mockdata". Изображения представлены в формате "data:URL".
